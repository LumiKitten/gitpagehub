# GitHub Pages Hub

A cozy portal for all my GitHub Pages projects.

## Features
- **Auto-Discovery**: Automatically fetches all repos with GitHub Pages enabled.
- **Priority Sorting**: Main projects are always pinned to the top.
- **'Featured' Tag support**: Add the `featured` topic to any repo on GitHub to push it higher.
- **Custom Branding**: Projects can have unique card background colors.
- **Dark Mode**: chill, easy-on-the-eyes dark pastel palette.
- **Micro-animations**: Smooth fade-ins and hover lift effects.
- **Dynamic Favicon**: Rounds my GitHub PFP as the site icon.

## Tech
- Vanilla HTML/CSS/JS
- GitHub REST API
- No frameworks, no dependencies.
